TypeScript

Advantages And Disadventages

Advantages

  • Strict typing

    Everything stays the way we define it. Need a variable to always be a number? It’ll always be a number, then.

  • Structural typing

    An approach where we look not at what the type is called or where it is defined, but at what it describes internally.

  • Early spotted bugs

    TypeScript features mistakes at arrangement time while JavaScript, at the runtime.

  • Rich IDE support

    Extraordinary tooling with IntelliSense which gives dynamic clues as the code’s additional.

  • Readability

    TypeScript makes the code more self-expressive

  • Power of OOP

Disadventages

  • False sense of security

    TS is transpiled into untyped JavaScript, so there is always the risk of weird type conversions at runtime.

  • Bloated code

    Has the potential of slowing down the development process.

  • No specification

    The last specification was on the version 1.8 / january 2016
